## 🔍 서비스 소개

> 감정 수확 일지, 감자전(傳) 
> 
> 감사한 일은 감자로, 속 답답한 일은 고구마로!
> 
> 감자와 고구마를 수확해 나만의 자서전(傳)을 완성해 보세요!
